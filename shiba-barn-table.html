SOUL STUDIO

Kids create one forever bond

Each child chooses a barn dog and mints one soulbound NFT: a permanent memory on the chain, never for trading, only for love. Privacy-first. Parent consent. Quality operators only.

The bond

  1. 1Meet barn souls with a handler. Calm. Safe. Real.
  2. 2The child chooses one dog that chose their heart back.
  3. 3Short “why I chose you” line + first name or nickname only.
  4. 4One soulbound recordⓘ. Permanent passport of the night.
  5. 5Small ceremony. You are 1. We are all 1.

Language is bond / memory / passport. Never investment, never flip. Aligns with the Unified Rescue Registry spirit of stories that cannot be sold away.

What lives on-chain

  • Dog name, barn, date of the Table
  • Child nickname (not full legal name)
  • Their short dedication line
  • Event id · Shiba Barn Table
  • Non-transferable · one per child per visit
